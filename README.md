# f1tenth

## Build
In catkin_ws/src:
```
git clone git@github.com:cavayangtao/f1tenth.git
cd ..
catkin_make
```

## Usage
1. Start a simulator
```
source devel/setup.bash
roslaunch f1tenth_simulator simulator.launch

```

2. Run!
```
source devel/setup.bash
rosrun f1tenth_simulator run_in_sim.py
```
