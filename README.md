# f1tenth

# Run in f1tenth_simulator

## Build
cd catkin_ws/src
```
git clone git@github.com:cavayangtao/f1tenth.git
cd ..
catkin_make
```
## Usage
1. Start a simulator

cd catkin_ws
```
source devel/setup.bash
roslaunch f1tenth_simulator simulator.launch

```
2. Run!

cd catkin_ws
```
source devel/setup.bash
rosrun f1tenth_simulator run_in_sim.py
```
